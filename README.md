## 📂 Featured Projects

### 🛒 [E-Commerce Sales Analysis — SQL + Python](https://github.com/IqraHasnain/SQL-Python-Ecommerce-Project)
> End-to-end analysis of 100K+ orders from a Brazilian e-commerce platform using SQL & Python.
- Answered 15 business questions across basic, intermediate, and advanced SQL
- Key finding: 20% YoY sales growth, $16M+ cumulative revenue, ~100% installment payment preference
- **Tools:** Python, Pandas, MySQL, Matplotlib, Seaborn, NumPy, Jupyter Notebook

---

### ☕ [Coffee Shop Sales Dashboard — Excel](https://github.com/IqraHasnain/Coffee-Shop-Sales)
> Interactive Excel dashboard analyzing 149K+ transactions across 3 store locations.
- Key finding: 9–10 AM peak hours, Barista Espresso = #1 product at $91K in sales
- Includes dynamic slicers for filtering by month and day
- **Tools:** Microsoft Excel, Pivot Tables, Pivot Charts
